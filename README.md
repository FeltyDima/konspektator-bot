# Konspektator Bot

Konspektator Bot — Telegram-бот для автоматической обработки учебных текстов.
Проект разработан в рамках технологической (проектно-технологической) практики.

Бот позволяет получать структурированные конспекты, определения терминов и упрощённые объяснения сложных тем на основе пользовательского текста.

---

## Возможности

- Создание структурированного конспекта учебного текста  
- Выделение ключевых терминов и их определений  
- Упрощённое объяснение сложных тем  
- Диалоговое взаимодействие через Telegram  
- Форматирование ответов для удобства чтения  

---

## Используемые технологии

- Python 3  
- Rasa Open Source (NLP и диалоговая логика)  
- Telegram Bot API  
- HTTP REST API  
- Railway (облачное развёртывание)  
- UptimeRobot (мониторинг доступности)

## Принцип работы

1. Пользователь отправляет учебный текст в Telegram  
2. Бот предлагает выбрать действие  
3. Запрос обрабатывается Rasa-сервером  
4. Результат форматируется и отправляется пользователю  
5. Пользователь может сразу отправить следующий текст  

---

## Развёртывание и мониторинг

Бот развёрнут на облачной платформе Railway, что обеспечивает круглосуточную работу.
Для контроля доступности сервиса используется инструмент UptimeRobot.
